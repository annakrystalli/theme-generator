# theme-generator
Xaringan presentation theme generator

- theme generator from template given parameters
- interactive shiny app add-in to help select styling with theme previews. 



## Template parameters

### slide
- bg_col 
- bg_img 


### text

- h1_col 
- h_col 
- body_col 
  
- h1_font 
- h_font 
- body_font 


## Generated parameters

- inv_bg_col = h_col 
- inv_font_col = bg_col 
- bg_col  
- inv_font
